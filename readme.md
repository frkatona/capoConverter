# Input chords and output valid capo positions
valid format:
 - capital letters alone for major chords (e.g., A major -> A)
 - capital letters with a lower case 'm' for minor chords (e.g., A minor -> Am)
 - separated by spaces (e.g., A Dm C Fm)
